Transparency at its core

Loving the Buffer policy of being transparent,i would try to explain my idea of doing this page.The kind of job i'll be doing section asked me to look at salaries page, and i figured out that on salaries page data was getting parsed into tabular form. I'd have loved to parse a json file and parse it into a tabular content using react as i have started getting myself familiar with react

But not having a json file at disposal , i used a JS library papa parse to parse a csv file representing data of Combined Land-Surface Air and Sea-Surface Water Temperature Anomalies (Land-Ocean Temperature Index, L-OTI) giving zonal annual means from 1880-2020.csv file obtained from here

Yes i could have written a json file myself but i was bored of working on fake self made json files after practsing few youtube react tutorials which used seld made json files so decided to use a csv file, and ended up learning new things in the process of tryin to implement
